（2）於「開始」-->「控制台」-->「系統」-->「進階」下，點選「系統環境變數」

（3）新增一個系統環境變數"SMLNJ_HOME"，並將變數值填入剛剛解壓縮的位置( D:\PL\smlnj )如下圖

（4）於PATH之下再加上一個D:\PL\smlnj\bin;的路徑 (如果之前PATH已經有東西的話，請使用分號區隔) 請不要把之前的Path刪除，以免造成系統不穩定

如果沒有PATH的話，請自行新增一個名為PATH的環境變數

（5）有兩種方法可以執行Standard ML of New Jersey（以後簡稱 SML）

1、到D:\PL\smlnj\bin之下，執行sml.bat，2、於「開始」-->「執行」中輸入sml，會出現如下的圖示：

（4）以下將透過簡易的範例，介紹兩種常用的Standard使用方式：

一、直接與 Standard ML of New Jersey（以後簡稱 SML）互動：

  • 當看到『-』符號時，表示可以開始使用 SML。

  • 接著依序輸入以下的範例（下圖有詳細說明）：

    fun inc x = x + 1 [Enter]

    fun f n = (inc n) * 2; [Enter]

    f (1+1); [Enter]

    • 二、藉由檔案與 SML 互動：

  • 將程式寫在一個『純文字檔』中，並假設檔名為test.sml。

  • 假設test.sml 的存放路徑為C:\sml\test.sml

  • 於 SML 輸入下列指令便可執行此程式（採用同樣的範例，如下圖所示）。

    use "C:\\SML\\test.sml"; [Enter](注意 ' \ ' 的數量)

    • 也可以將.sml的檔案全部設成自動由D:\PL\smlnj\bin\sml.bat執行。
